포켓몬 랜덤 도감 뽑기

위에서 전국도감 또는 각 버전(DP, Pt, HGSS, BW, BW2, XY, ORAS, SM, USUM, LPLE, SwSh, BDSP, SV)을 선택한 뒤
“포켓몬 뽑기”를 누르면 해당 기준에 맞춰 랜덤으로 한 마리를 뽑습니다.
기준 선택
데이터 입력 위치 안내 (자기가 직접 채우는 구역)
1) 전국도감: pokemonNames 안에 번호: '포켓몬 이름' 형식으로 1~1025를 채우면, 전국도감 모드에서 사용됩니다.
2) 버전별 도감: dexLists 안에서 버전별로 같은 형식으로 채우면, 해당 버전 모드에서 사용됩니다.
